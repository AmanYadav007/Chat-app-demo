# Aguider Web App

## Description

Welcome to Aguider, your personal space for open communication and support. This Chrome extension enables users to chat with the admin (you) and share their feelings, thoughts, or concerns in a secure environment.

## Features

- **Chatting:** Engage in real-time conversations with users.
- **Feelings Sharing:** Users can express their emotions and thoughts with the admin.
- **Admin Interface:** Exclusive access to an admin interface for managing and responding to user messages.

## Installation

<span style="color: green">Follow these steps to install the Aguider extension:</span>

1. Clone this repository to your local machine.
   ```bash
   git clone https://github.com/AmanYadav007/AGuider.git
   ```

Open Chrome and go to chrome://extensions/.

Enable "Developer mode" in the top right corner.

Click on "Load unpacked" and select the directory where you cloned the Aguider extension.

The extension is now added to Chrome.

## Usage
Click on the Aguider extension icon in the Chrome toolbar.

Log in as the admin using your credentials.

Start chatting with users and provide support or engage in conversations.

## Admin Interface
To access the admin interface:

Log in with your admin credentials.

Navigate to the admin dashboard where you can view and respond to user messages.
